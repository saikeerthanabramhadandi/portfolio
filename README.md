# 🚀 FUTURE_FS_01 - Personal Portfolio Website

A modern, responsive personal portfolio website built using React.js and Tailwind CSS.  
This project showcases my skills, projects, coding profiles, education, and contact system with a clean UI and smooth user experience.

---

## 👩‍💻 About the Project

This portfolio is developed as part of my internship/task requirement.  
It represents my frontend development skills and includes real-world features like project showcase, coding profiles, education details, certificate viewing, and email integration.

---

## 🌟 Features

- Fully responsive design (Mobile + Desktop)
- Modern UI with Tailwind CSS styling
- Animated hero section using react-type-animation
- Sticky navbar with smooth navigation
- About section with personal introduction
- Skills showcase section
- Education section (B.Tech - Aditya College of Engineering and Technology)
- Journey timeline section
- Coding profiles section (LeetCode, HackerRank, CodeChef)
- Projects section with GitHub + Live links
- Certificate viewer (PDF opens in browser)
- Contact form with EmailJS integration (real email sending)
- Smooth scroll navigation between sections
- Card-based UI with hover animations

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- EmailJS (@emailjs/browser)
- React Type Animation
- React Icons
